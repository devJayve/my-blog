# Understanding TypeScript: A Beginner's Guide

TypeScript is a superset of JavaScript that adds static typing to the language. It is designed to help developers write safer and more reliable code by catching errors at compile time rather than at runtime.

## Why TypeScript?

Using TypeScript can greatly improve the development experience by providing:

- **Static Type Checking:** TypeScript checks for type-related errors during the development process.
- **Improved IDE Support:** With TypeScript, editors can offer better autocomplete, refactoring, and navigation features.
- **Enhanced Code Quality:** By enforcing type constraints, TypeScript encourages more robust and maintainable code.

Let's start with a basic example of TypeScript in action. Here’s how you would write a simple TypeScript function:
